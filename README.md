\# Tableau Public Dashboards



This repository stores evidence and documentation for data dashboards created using Tableau Public.



\## Purpose

Show data visualization and reporting skills through interactive Tableau dashboards.



\## What This Repo Contains

\- Links to dashboards hosted on Tableau Public

\- Images and screenshots of visual outputs

\- Descriptions of datasets used

\- Explanations of insights from each dashboard



\## Focus

Demonstrate ability to:

\- transform raw data into visuals

\- build KPIs

\- communicate trends clearly
